# LinkedIn Engagement Analysis

## Project Structure
This project is structured to analyze LinkedIn post performance using the following pipeline:
1. Data Cleaning (`src/data_cleaning.py`)
2. Feature Engineering (`src/feature_engineering.py`)
3. Modeling (`src/models.py`)

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Place your raw JSON data in `data/raw/`
3. Run `python pre_clean_data.py` to generate the CSV.
